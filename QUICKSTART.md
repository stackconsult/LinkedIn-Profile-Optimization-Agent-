# Quick Start Guide

## Getting Started in 3 Steps

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Set Up Your API Key
Create a `.env` file:
```bash
echo "OPENAI_API_KEY=your_api_key_here" > .env
```

Or enter it directly in the app when prompted.

### 3. Run the App
```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`

## What You Can Do

1. **Analyze Your Headline**
   - Enter your current LinkedIn headline
   - Select your target audience (jobs, clients, etc.)
   - Get AI-powered feedback and suggestions

2. **Optimize Your About Section**
   - Paste your current About section
   - Receive scored feedback (0-10)
   - Get specific improvement suggestions

3. **Learn Best Practices**
   - Review general LinkedIn optimization tips
   - Understand what makes profiles stand out
   - Apply proven strategies to your profile
